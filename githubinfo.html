All about GitHub

  • Q. What is a version control system?
  • A. A software that automatically tracks any change made to files or code
  • Q. What is Git?
  • A. A common version control system developed by Linus Torvalds
  • Q. What is GitHub?
  • A. A website hosting service used to store Git Versions and Repositories.
  • Q. What is the difference between Git and GitHub?
  • A. Git is a version control software, GitHub is a website used to store and manage version controlled software
  • Q. Who started GitHub and how was it started?
  • A. Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, Scott Chacon. Made using Ruby
  • Q. What company owns it now?
  • A. Microsoft
  • Q. How much does a GitHub account cost?
  • A. Outside of premium memberships, they are free.
  • Q. What is the Octocat?
  • A. The Trademarked "Mascot" of github.
