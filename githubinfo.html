All about GitHub

  1. Q. What is a version control system? A. A software that automatically tracks any change made to files or code
  2. Q. What is Git? A. A common version control system developed by Linus Torvalds
  3. Q. What is GitHub? A. A website hosting service used to store Git Versions and Repositories.
  4. Q. What is the difference between Git and GitHub? A. Git is a version control software, GitHub is a website used to store and manage version controlled software
  5. Q. Who started GitHub and how was it started? A. Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, Scott Chacon. Made using Ruby
  6. Q. What company owns it now? A. Microsoft
  7. Q. How much does a GitHub account cost? A. Outside of premium memberships, they are free.
  8. Q. What is the Octocat? A. The Trademarked "Mascot" of github.
