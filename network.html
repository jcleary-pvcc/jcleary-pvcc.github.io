Chapter 4 Networking Terminology

Term Definition
Network Classifications
  • LAN
  • MAN
  • WAN
  • Local Area Network; used for closely connected devices.
  • Metropolitan Area Network; A network that spans a city.
  • Wide Area Network; Generally spans a large area, often switched with "internet"
Network Models
  • bus network
  • star network
  • A network connected in a direct line or "bus"
  • A network centering on a single device/controller, forming a "star" on a map.
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
  • ethernet procotol that dictates when a packet is sent on a bus.
  • Link Layer protocol used when there is a WIFI connection.
Combining networks
  • repeater
  • bridge
  • switch
  • router
  • gateway
  • used to "repeat" a message to increase its distance
  • a device that connects two seperate networks.
  • (switch) Networking device that replaced Hubs, allowing a packet to be sent only to its intended recipient.
  • Routers allow a network to send packets outside its network.
  • allows one to connect to a network from outside it.
Client/server model
  • P2P: peer-to-peer
  • distributed system
  • cloud computing
  • P2P directly connects two devices, no middle man.
  • A system where processing is split between multiple computers.
  • A model where all processing power is offloaded to an external service, made available through the internet
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
  • T1: the major companies that are the "backbone" of the internet. I.E: ICANN, AT&T
  • T2: Smaller companies that provide local internet services.
  • T3: The stage before the end-user. Often the company providing the internet/routing.
IP Address
  • ICANN
  • DNS
  • TLD: Top-level Domain
  • Country-code TLDs
  • The organization every company must register with to provide domains.
  • The Domain Name System, used to provide a "name" for IP links. I.E www.google.com
  • the most common suffixes for a website. I.E: .edu, .gov, .com
  • The TLD suffix for where the website is based. I.E .us for united states.
Internet Protocol
  • FTP
  • HTTP
  • SMTP
  • File Transfer Protocol: sends files through the internet to their intended port.
  • Hypertext Transfer Protocol. Used to tranfer data across the internet.
  • Simple Mail Transfer Protocol: A common protocol used to send Email.
World Wide Web
  • URL
  • client-side activities
  • server-side activities
  • Uniform Resource Locator: used to provide the "links" to a location on a network.
  • Take place on the "client" computer.
  • Take place on the "server" the website is hosted on.
Internet four-layer software
  • application layer
  • transport layer
  • network layer
  • link layer
  • Handles the interfaces and other "user" sections. 4th layer.
  • Ensures the integrity of sent data. 3rd layer.
  • Manages the networking of the packets. 2nd layer.
  • Uses computer features such as the MAC address to manage connections. 1st layer.
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
  • A common networking method where sent data is arranged into "packets"
  • Able to take faults/breaks in the network.
  • Extra/excess parts of a network
Cybersecurity
  • malware
  • virus
  • worm
  • Trojan horse
  • spyware
  • phishing
  • DDoS attack
  • A malicious software or package.
  • A malicious string of code that "infects" other programs.
  • A malicious file that transfers over networks.
  • malware that disguises itself as a normal piece of software.
  • Malware that "spys" on the user's data.
  • A cyber-attack that attempts to steal a user's credentials.
  • A cyber-attack that spams a website with many garbage connections from multiple bot computers.
Protection
  • firewall
  • spam filter
  • proxy server
  • antivirus software
  • cryptography
  • public key encryption
  • Blocks specific packets from going in or out.
  • A feature which blocks "spam" or repeated messages.
  • A server that sends/retrieves packets on behalf of a client.
  • Software dedicated to protecting from malicious software. I.E kaspersky.
  • The study of creating codes and encryption.
  • A commonly used type of encryption that encrypts a message using "public" and "private" key files.
